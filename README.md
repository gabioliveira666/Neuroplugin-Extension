# Extensão de Acessibilidade para Neurodivergentes

Esta é uma extensão para o navegador Google Chrome focada em melhorar a acessibilidade e a experiência de leitura para pessoas neurodivergentes, como aquelas com TDAH, dislexia e autismo, e para pessoas com diferentes tipos de daltonismo.

## Funcionalidades

A extensão permite aplicar diferentes modos de visualização e leitura em qualquer site, incluindo:

- **Modo Destacado:** destaca palavras curtas para auxiliar na leitura rápida.
- **Modo Espaçado:** aumenta o espaçamento entre letras e palavras para facilitar a leitura.
- **Modo Simplificado:** altera a tipografia e espaçamento do texto para uma leitura mais clara e objetiva.
- **Modo Claro:** fundo branco e texto escuro para alto contraste.
- **Modo Escuro:** fundo escuro e texto claro para reduzir o cansaço visual.
- **Filtros de Daltonismo:**
  - **Protanomalia**
  - **Deuteranomalia**
  - **Tritanomalia**

## Inclusividade

Esta extensão foi criada com base em princípios de design inclusivo, com foco em:

- Redução da sobrecarga sensorial
- Melhor processamento visual
- Suporte a diferentes perfis de percepção de cor

## Tecnologias utilizadas

- JavaScript
- HTML
- CSS
- Chrome Extensions API

## Como usar

1. Instale a extensão no Chrome via modo de desenvolvedor.
2. Clique no ícone da extensão para abrir o popup.
3. Ative os modos desejados por meio dos botões.
4. Aplique estilos que melhor se adaptam à sua leitura.

## Passo a passo da instalação Manual

1. Baixe ou clone este repositório.
2. No Chrome, vá até `chrome://extensions/`.
3. Ative o **Modo do desenvolvedor** no canto superior direito.
4. Clique em **Carregar sem compactação** e selecione a pasta da extensão.
5. A extensão aparecerá na barra de ferramentas.
